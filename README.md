# CONVERSOR-DE-DOLAR-E-REAL
Neste projeto, que foi realizado o VS code (com a linguagem python), é um código que converte o valor de Dólar para real e, vice-versa, de Real para Dólar. Quando o usuário, escolhe alguma opção, ou seja, qual conversão ele deseja, após isso, ele insere o valores e o código executa a conversão.
